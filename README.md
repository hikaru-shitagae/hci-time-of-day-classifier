# HCI 前半レポート課題

都市/屋外風景画像から時間帯（day / night / sunrise_sunset）を推定する画像分類。

## Project structure
- `data/` : datasets (ignored by git)
- `notebooks/` : Jupyter notebooks
- `src/` : training / evaluation scripts (optional)
- `models/` : saved weights (ignored by git)
- `outputs/` : logs, figures (ignored by git)

## Dataset
- Kaggle: `aymenkhouja/timeofdaydataset`
