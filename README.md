# HCI 前半レポート課題概要

屋外風景画像から時間帯を推定する3クラス分類器を構築し，
都市/屋外風景画像から時間帯（day / night / sunrise_sunset）を推定する画像分類を行うプログラムである．

## 使用したプロンプト
- kaggleの`aymenkhouja/timeofdaydataset`を用いて時間帯（day / night / sunrise_sunset）を推定するプログラムの仕様書を作成するようにChatGPTに依頼
- 得られた仕様書を自分で編集し，`specification.md`として保存し，そのファイルをGitHub Copilotに読み込ませることで，リポジトリにあるファイル・プログラムを一括作成

---
